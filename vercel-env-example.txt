# Vercel Environment Variables 配置指南

## 在 Vercel Dashboard 中添加以下环境变量：

### 数据库配置
DATABASE_URL=postgresql://postgres:Supabase123!@db.efzpntcevdiwkaqubrxq.supabase.co:5432/postgres?sslmode=require

### NextAuth 配置（可选）
NEXTAUTH_SECRET=your-secret-key-here
NEXTAUTH_URL=https://your-app.vercel.app

### 其他可能需要的配置
NODE_ENV=production

## 配置步骤：
1. 登录 Vercel Dashboard
2. 选择您的项目
3. 进入 Settings → Environment Variables
4. 点击 Add New
5. 输入 Key 和 Value
6. 选择 Environment: Production, Preview, Development
7. 点击 Save

## 重新部署：
配置完环境变量后，需要重新部署项目来应用新的配置。
